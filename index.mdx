欢迎来到 **AI 办公助手教程**。这是一门专为业务人员设计的实战课程，帮助你用 AI 工具高效完成日常工作。

<Note>
  **面向对象**：业务人员（非技术背景，不需要会写代码、用终端或 git）

  **课程目标**：学完能用 VS Code + Claude Code 插件完成「读文档、问问题、改文案、整理信息」

  **建议时长**：90 分钟（含 15 分钟动手练习）
</Note>

## 学完你能做什么

<CardGroup cols={2}>
  <Card title="读懂任何文档" icon="book-open">
    技术文档、配置文件、数据表——让 Claude 用你听得懂的话解释
  </Card>
  <Card title="写结构化文档" icon="pen-to-square">
    PRD、FAQ、发布说明、变更记录、周报材料——从口头需求到正式文档
  </Card>
  <Card title="改写文案" icon="wand-magic-sparkles">
    调整语气、统一术语、生成多版本——一段原文变出多个版本
  </Card>
  <Card title="整理和分析数据" icon="table">
    CSV、Excel 数据概况、异常值检测、关键发现——快速提炼信息
  </Card>
</CardGroup>

## 课程大纲

<Steps>
  <Step title="认识 VS Code（15 min）">
    了解为什么用 VS Code，学会打开文件夹、认识界面关键区域
  </Step>
  <Step title="安装并使用 Claude Code 插件（20 min）">
    安装插件、登录验证、掌握四种核心用法
  </Step>
  <Step title="业务场景实战（25 min）">
    五个真实业务场景：读文档、写文档、查一致性、改文案、分析数据
  </Step>
  <Step title="提问技巧与交互节奏（15 min）">
    掌握黄金节奏、提问三原则、高效话术模板
  </Step>
  <Step title="Agent Skills 介绍（10 min）">
    把 SOP 变成 Claude 的"技能包"，了解落地路径
  </Step>
  <Step title="进阶了解（5 min）">
    CLAUDE.md 规则文件、多会话管理、与技术同学的协作分工
  </Step>
  <Step title="Q&A + 动手练习（15 min）">
    四个练习任务，从入门到综合应用
  </Step>
</Steps>
