以下只列出了业务人员日常工作中最常用的快捷键，不需要全部记住，用到的时候查一下就行。

## 文件操作

| 操作 | Mac | Windows |
|------|-----|---------|
| 打开文件夹 | `Cmd + O` → 选文件夹 | `Ctrl + K` 然后 `Ctrl + O` |
| 搜索文件 | `Cmd + P` | `Ctrl + P` |
| 保存文件 | `Cmd + S` | `Ctrl + S` |
| 撤销 | `Cmd + Z` | `Ctrl + Z` |
| 重做 | `Cmd + Shift + Z` | `Ctrl + Y` |

## VS Code 操作

| 操作 | Mac | Windows |
|------|-----|---------|
| 打开扩展市场 | `Cmd + Shift + X` | `Ctrl + Shift + X` |
| 打开命令面板 | `Cmd + Shift + P` | `Ctrl + Shift + P` |
| 切换侧边栏 | `Cmd + B` | `Ctrl + B` |

## Claude Code 操作

| 操作 | Mac | Windows |
|------|-----|---------|
| 插入 @ 引用 | `Option + K` | `Alt + K` |
| 新建 Claude 对话 | `Cmd + N` | `Ctrl + N` |

<Tip>
  **最重要的三个快捷键**：
  1. `Ctrl + S`（保存）—— 修改文件后记得保存
  2. `Ctrl + Z`（撤销）—— 改错了随时撤销
  3. `Ctrl + P`（搜索文件）—— 找不到文件时用这个
</Tip>
