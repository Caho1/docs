以下话术模板经过实际验证，可以直接复制到 Claude 对话框中使用。只需要把 `[方括号]` 里的内容替换成你的实际信息。

## 通用总结模板

```
请阅读 @[文件名]，用以下格式总结：
- 一句话概括（不超过 30 字）
- 3-5 条核心要点
- 需要我关注的风险/待确认事项
```

**适用场景**：快速了解任何文档的核心内容

## 文档改写模板

```
请把 @[文件名] 中的 [具体部分] 改写为 [目标风格/格式]。
要求：
1. 保留所有关键信息，不要遗漏
2. [具体约束，如字数限制、术语要求等]
3. 改完后列出你做了哪些改动
```

**适用场景**：调整文档语气、风格、面向的读者

## 信息提取模板

```
请从 @[文件名] 中提取以下信息，整理成表格：
列：[列 1]、[列 2]、[列 3]
如果某项信息缺失，标注"未提及"。
```

**适用场景**：从长文档中提炼结构化信息

## 对比分析模板

```
请对比 @[文件A] 和 @[文件B]：
1. 两份文件的主要差异有哪些？
2. 有没有互相矛盾的地方？
3. 用表格列出差异项
```

**适用场景**：比较两个版本的文档、对比需求和设计方案

## 邮件生成模板

```
请帮我写一封邮件：
- 收件人：[角色/关系]
- 目的：[要传达什么]
- 语气：[正式/友好/紧急]
- 需要对方做什么：[具体行动]
```

**适用场景**：快速写工作邮件

## 会议纪要模板

```
请根据 @[会议笔记文件] 整理成正式的会议纪要：
- 会议基本信息（时间、参会人）
- 讨论要点（按议题分条）
- 决议事项
- Action Items（负责人 + 截止日期）
- 遗留问题
```

**适用场景**：把零散的会议笔记整理成结构化的纪要
