## claude.ai 和 VS Code + Claude Code 的区别

你可能会问：既然有 claude.ai 网页版，为什么还要装 VS Code？

核心区别在于：**claude.ai 是"聊天"，Claude Code 是"协作"**。

| | claude.ai 网页版 | VS Code + Claude Code 插件 |
|---|---|---|
| **工作方式** | 你把内容复制粘贴给它 | 它直接读取你电脑上的文件 |
| **修改文件** | 它给你建议，你手动改 | 它直接帮你改，你看对比后决定接受/拒绝 |
| **上下文** | 只知道你贴给它的内容 | 能看到整个文件夹的结构和内容 |
| **适合场景** | 快速问答、闲聊 | 处理文件、改文档、整理信息 |

<Tip>
  **一句话总结**：当你的工作涉及具体的文件和文档时，VS Code + Claude Code 插件是效率更高的选择。
</Tip>

## 举个例子

假设你需要检查一份 10 页的产品方案文档：

**用 claude.ai**：你需要把整份文档复制粘贴到聊天框 → Claude 给出建议 → 你再手动回到文档里一个个改。

**用 VS Code + Claude Code**：你直接打开文件夹 → 告诉 Claude "帮我检查这份方案" → Claude 直接在文件上标出问题并提供修改建议 → 你逐条确认接受或拒绝。

效率差距一目了然。

## 安装 VS Code

安装非常简单：

<Tabs>
  <Tab title="Windows">
    从公司软件中心或 [VS Code 官网](https://code.visualstudio.com/) 下载 `.exe` 安装包，双击安装。安装过程中一路点"下一步"即可。
  </Tab>
  <Tab title="macOS">
    从 [VS Code 官网](https://code.visualstudio.com/) 下载 `.dmg` 文件，双击打开后将 VS Code 图标拖入 Applications 文件夹。
  </Tab>
</Tabs>

安装完成后，打开 VS Code，你会看到一个欢迎页面——说明安装成功了。
