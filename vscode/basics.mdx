## 打开文件夹（不是打开文件）

<Warning>
  **新手最常犯的错误**：打开了单个文件，而不是整个文件夹，导致看不到完整的项目结构。Claude Code 也无法看到你文件夹里的其他文件。
</Warning>

正确操作：

<Steps>
  <Step title="打开文件夹">
    菜单栏 → `File`（文件）→ `Open Folder...`（打开文件夹）
  </Step>
  <Step title="选择目录">
    选择你要工作的目录（比如课程练习文件夹）
  </Step>
  <Step title="确认成功">
    看到左侧出现**文件树**（Explorer），说明打开成功
  </Step>
</Steps>

输入文件名的一部分就能快速定位。

## 认识界面四个关键区域

打开文件夹后，你会看到 VS Code 的主界面。记住这四个区域：

<Frame>
  ![Image](/images/image-7.png)
</Frame>

<CardGroup cols={2}>
  <Card icon="folder-tree" title="左侧：文件树">
    你所有的文件和文件夹都在这里，点击就能打开对应的文件
  </Card>
  <Card icon="file-lines" title="中间：编辑区">
    查看和编辑文件内容的地方，打开的文件会以标签页形式显示在顶部
  </Card>
  <Card icon="message-bot" title="右侧：Claude 面板">
    安装 Claude Code 插件后出现，这是你和 Claude 对话的地方
  </Card>
  <Card icon="terminal" title="底部：终端">
    暂时不用关注，技术同学会用到这里
  </Card>
</CardGroup>

<Tip>
  在日常使用中，你主要关注**左侧文件树**（找文件）、**中间编辑区**（看内容）和**右侧 Claude 面板**（跟 Claude 对话）就够了。
</Tip>
