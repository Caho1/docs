## 你不需要会什么

在开始之前，请放心——这门课程**不需要任何技术背景**：

- 不需要会写代码
- 不需要会用终端 / 命令行
- 不需要会用 git
- 不需要理解任何编程概念

你只需要会用电脑、会打字、会上网就够了。

请在课程开始前完成以下准备：

## 练习文件夹说明

课程使用一个**非代码**的 demo 文件夹，包含以下类型的文件：

| 文件类型 | 说明 | 示例 |
|---------|------|------|
| Markdown 文档 | 纯文本格式的文档，可以直接阅读和编辑 | `README.md`、`meeting-notes.md` |
| JSON 配置文件 | 结构化的配置信息 | `config.json` |
| CSV 数据表 | 用逗号分隔的表格数据 | `data.csv` |

这些文件都是模拟的业务场景数据，让你在真实文件上操作，而不是对着代码发呆。
