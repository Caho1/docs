使用课程提供的练习文件夹，完成以下四个练习任务。

## 练习 1：读懂文档（入门）

**任务**：打开 `README.md`，让 Claude 用非技术语言总结这份文档。

**参考话术**：
```
请读一下 @README.md，用我能理解的话告诉我：
1. 这个项目是做什么的
2. 有哪些核心功能
3. 有什么注意事项
```

**考察点**：能打开文件夹、能和 Claude 对话、能理解回答

## 练习 2：改写文案（基础）

**任务**：打开 `announcement-draft.md`，选中正文部分，让 Claude 生成正式版和口语版两个版本。

**参考话术**：
```
请把我选中的内容改写成两个版本：
1. 正式版：适合发公告邮件
2. 口语版：适合发在团队群里
保留所有关键信息。
```

**考察点**：会使用"选中文本 + 提问"的方式

## 练习 3：分析数据（进阶）

**任务**：打开 `data.csv`，让 Claude 分析数据概况，找出异常值，并用表格呈现。

**参考话术**：
```
请读取 @data.csv，告诉我：
1. 数据有多少行、多少列、每列是什么意思
2. 有没有异常值或空值
3. 你发现的 3 个关键信息
用表格呈现结果。
```

**考察点**：会用 @ 引用文件、能提出有约束的需求

## 练习 4：整理会议纪要（综合）

**任务**：根据 `meeting-notes.md` 的内容，让 Claude 按照标准格式重新整理。

**参考话术**：
```
请读取 @meeting-notes.md，按以下格式整理成正式会议纪要：
- 会议信息（时间、地点、参会人）
- 讨论要点（按议题分条）
- 决议事项
- Action Items（负责人 + 截止日期）
如有信息缺失，标注"待补充"。
```

**考察点**：综合运用所有技巧——引用文件、格式要求、约束条件

## 评估标准

| 评估维度 | 说明 |
|---------|------|
| 能打开文件夹并使用 Claude | VS Code 基础操作 |
| 会用"选中文本 + 提问" | 最核心的交互方式 |
| 提问有明确目标和约束 | 不是"帮我看看"，而是具体要求 |
| 能接受/拒绝 Claude 的修改 | 掌握 Diff 视图操作 |
| 知道跑偏时怎么处理 | 开新对话或拒绝修改 |

<Tip>
  不要怕犯错。练习的目的就是让你熟悉操作。如果遇到问题，随时问讲师或查看[常见问题排查](/claude-code/troubleshooting)。
</Tip>
