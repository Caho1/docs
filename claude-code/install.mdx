## 从扩展市场安装

<Steps>
  <Step title="打开扩展市场">
    点击左侧边栏的**四个小方块图标**（Extensions），或使用快捷键：
    - **Mac**：`Cmd + Shift + X`
    - **Windows**：`Ctrl + Shift + X`
  </Step>
  <Step title="搜索插件">
    在搜索框输入 **Claude Code**
  </Step>
  <Step title="认准官方插件">
    找到 **Anthropic** 发布的官方插件（注意认准发布者名称，避免安装第三方仿冒插件）
  </Step>
  <Step title="点击安装">
    点击 **Install**（安装），等待安装完成
  </Step>
</Steps>

## 登录与验证

安装完成后，需要登录你的 Claude 账号：

<Steps>
  <Step title="打开 Claude 面板">
    侧边栏会出现一个 **Claude 图标**（星形图标），点击打开对话面板。

    如果面板没出现，可以打开命令面板（`Cmd+Shift+P` / `Ctrl+Shift+P`），输入 "Claude Code"，选择 **Open in New Tab** 或 **Open in Sidebar**。
  </Step>
  <Step title="登录账号">
    首次使用会提示登录——点击 `Sign in`，按提示在弹出的浏览器页面完成授权。
  </Step>
  <Step title="测试连接">
    回到 VS Code，发一条测试消息（比如"你好"），收到回复说明安装成功。
  </Step>
</Steps>

<Tip>
  底部状态栏也会出现 "Claude Code" 字样，点击它同样可以打开对话面板。
</Tip>

## 打开 Claude 面板的三种方式

安装完成后，有多种方式打开 Claude 对话面板，选择你觉得顺手的即可：

<Tabs>
  <Tab title="方式一：右上角图标">
    安装完成后，VS Code 主界面的**右上角**会出现 Claude Code 的图标，点击即可打开。

    打开后 Claude 会以**标签页**的形式显示在编辑区。如果你觉得标签页太小，可以用鼠标**拖动标签页**到你想要的位置，比如拖到编辑区让它全屏显示。
  </Tab>
  <Tab title="方式二：侧边栏">
    点击左侧边栏的 **Claude 图标**（星形图标），Claude 面板会在侧边栏中打开。

    这种方式适合一边看文件、一边和 Claude 对话的场景。
  </Tab>
  <Tab title="方式三：命令面板">
    打开命令面板（`Cmd+Shift+P` / `Ctrl+Shift+P`），输入 "Claude Code"，选择：
    - **Open in New Tab**：在新标签页中打开（更大的对话空间）
    - **Open in Sidebar**：在侧边栏中打开（方便同时查看文件）
  </Tab>
</Tabs>

## 权限模式说明

Claude Code 有不同的权限模式，控制 Claude 在操作文件时是否需要你手动确认：

| 模式 | 说明 | 适合谁 |
|------|------|--------|
| **默认模式** | Claude 每次修改文件前都会询问你确认 | 新手推荐，每一步都在掌控中 |
| **Auto-accept 模式** | Claude 的修改会自动生效，不需要逐条确认 | 熟练后使用，效率更高 |

切换方式：在 Claude 对话面板中按 `Shift + Tab` 可以快速切换权限模式。

<Info>
  **新手建议**：刚开始使用时保持默认模式，让 Claude 每次修改都征求你的同意。等你熟悉了 Claude 的工作方式、对它的输出质量有信心之后，再考虑切换到 Auto-accept 模式提升效率。
</Info>

## 安装验证清单

确认以下几项都没问题：

| 检查项 | 预期结果 |
|--------|---------|
| 侧边栏有 Claude 图标 | 能看到星形图标 |
| 点击图标后出现对话面板 | 面板显示输入框 |
| 发送"你好"能收到回复 | Claude 正常回复中文 |
| 底部状态栏有 Claude Code | 显示连接状态 |

如果以上任何一项不正常，请参考[常见问题排查](/claude-code/troubleshooting)。
