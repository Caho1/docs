## 从扩展市场安装

<Steps>
  <Step title="打开扩展市场">
    点击左侧边栏的**四个小方块图标**（Extensions）
  </Step>
  <Step title="搜索插件">
    在搜索框输入 **Claude Code**
  </Step>
  <Step title="认准官方插件">
    找到 **Anthropic** 发布的官方插件（注意认准发布者名称，避免安装第三方仿冒插件）
  </Step>
  <Step title="点击安装">
    点击 **Install**（安装），等待安装完成
  </Step>
</Steps>

## 方法二

### 打开/创建：`~/.claude/settings.json`

```text
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "替换为下面的API Key",
    "ANTHROPIC_BASE_URL": "https://dm-fox.rjj.cc/claude/droid",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1，
    "ANTHROPIC_MODEL": "claude-sonnet-4-6",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "claude-sonnet-4-6"
  },
  "permissions": {
    "allow": [],
    "deny": []
  }
}
```

> **可用这个 key👇 来试用 ClaudeCode，默认模型 Claude Sonnet-4.6**

## 打开 Claude 面板的三种方式

安装完成后，有多种方式打开 Claude 对话面板，选择你觉得顺手的即可：

<Tabs>
  <Tab title="方式一：右上角图标">
    打开 Claude 的最快方式是点击**编辑器工具栏**（编辑器右上角）中的 Spark 图标。当你打开文件时，该图标才会出现。

## 权限模式说明

Claude Code 有不同的权限模式，控制 Claude 在操作文件时是否需要你手动确认：

| 模式                 | 说明                      | 适合谁           |
| ------------------ | ----------------------- | ------------- |
| **默认模式**           | Claude 每次修改文件前都会询问你确认   | 新手推荐，每一步都在掌控中 |
| **Auto-accept 模式** | Claude 的修改会自动生效，不需要逐条确认 | 熟练后使用，效率更高    |

切换方式：在 Claude 对话面板中按 `Shift + Tab` 可以快速切换权限模式。

<Info>
  **新手建议**：刚开始使用时保持默认模式，让 Claude 每次修改都征求你的同意。等你熟悉了 Claude 的工作方式、对它的输出质量有信心之后，再考虑切换到 Auto-accept 模式提升效率。
</Info>
