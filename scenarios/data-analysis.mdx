## 背景

你有一份 CSV 或 Excel 数据，需要快速了解数据概况或提炼关键信息。以前你可能需要打开 Excel 手动筛选和统计，现在可以让 Claude 帮你做。

## 推荐话术

```
请读取 @data.csv，帮我完成：
1. 数据概况：一共多少行、多少列，每列的含义
2. 数据质量：有没有空值、异常值、重复行
3. 关键发现：你觉得值得关注的 3-5 个发现
4. 请用表格形式呈现结果
```

## 更多数据分析话术

<Tabs>
  <Tab title="数据汇总">

    ```
    请读取 @sales-data.csv，按以下维度汇总：

    2. 按地区统计订单量
    3. 找出销售额最高和最低的产品
    ```

  </Tab>
  <Tab title="数据对比">

    ```
    请对比 @本月数据.csv 和 @上月数据.csv：
    1. 哪些指标有明显变化？（增减超过 10%）
    2. 有没有新增或消失的项目？
    3. 用表格列出变化最大的前 5 项
    ```
  </Tab>
  <Tab title="异常检测">
    ```
    请检查 @data.csv 中的数据质量：
    1. 有没有明显不合理的数值？（比如负数、超大值）
    2. 有没有格式不统一的数据？（比如日期格式混乱）
    3. 有没有空白或缺失的必填字段？
    列出所有问题，标注在第几行
    ```
  </Tab>
</Tabs>

## 使用技巧

<Note>
  **关于数据量**：Claude 可以处理几千行的 CSV 数据没有问题。如果数据特别大（超过 1 万行），建议分段处理，或者先让 Claude 做抽样分析。
</Note>

<Warning>
  **关于数据准确性**：Claude 的数据分析结果应该作为参考，重要的数据决策请用 Excel 或专业工具二次验证。Claude 偶尔会在数学计算上出错。
</Warning>

### 让 Claude 帮你整理输出

分析完数据后，你可以进一步要求：

```
把刚才的分析结果整理成一份简报，我需要发给老板。
要求：
- 用 3-5 条要点概括
- 每条要点配一个关键数字
- 最后附上你的建议（1-2 条）
```
