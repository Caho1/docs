## 背景

你有一段现成的文案，需要根据不同场景调整风格。比如同一个通知，给领导的邮件要正式，发团队群要口语化，做标题要精炼。

## 操作方式

这个场景最适合使用**"选中文本再提问"**的方式：

<Steps>
  <Step title="打开文件">
    打开包含你要改写内容的文件
  </Step>
  <Step title="选中文本">
    用鼠标拖选你要改写的那段文字
  </Step>
  <Step title="输入需求">
    在 Claude 对话框输入改写要求
  </Step>
  <Step title="审核结果">
    查看 Claude 给出的多个版本，选择最合适的
  </Step>
</Steps>

## 推荐话术

### 多风格改写

```
请把我选中的这段内容改写成三个版本：
1. 正式版：适合发公告或写邮件给领导
2. 口语版：适合在团队群里通知大家
3. 简洁版：控制在 50 字以内，适合做标题或摘要

保留所有关键信息，不要添加原文没有的内容。
```

### 语气调整

```
请把选中的这段文字改成更 [友好/专业/紧急/委婉] 的语气，
保持原意不变。
```

### 面向不同读者

```
请把选中的内容分别改写成面向以下读者的版本：
1. 面向客户：专业友好，突出价值
2. 面向内部团队：直接简洁，突出行动项
3. 面向管理层：概括性强，突出影响和结论
```

## 实际案例

假设你选中了这段原文：

> 系统将于本周五 18:00-22:00 进行升级维护，届时报表导出功能将暂时不可用。本次升级新增了 Excel 和 PDF 导出格式，并修复了筛选条件偶发失效的问题。

Claude 可能会输出：

<Tabs>
  <Tab title="正式版">
    > 尊敬的用户，我们计划于本周五（18:00-22:00）对系统进行升级维护。维护期间，报表导出功能将暂时停止服务。本次升级将新增 Excel 及 PDF 导出格式，同时修复已知的筛选条件异常问题。给您带来的不便，敬请谅解。
  </Tab>
  <Tab title="口语版">
    > 周五晚上 6 点到 10 点系统要升级，这段时间报表导出用不了哈。升完级之后就能导出 Excel 和 PDF 了，之前筛选偶尔不好使的问题也会修好。
  </Tab>
  <Tab title="简洁版">
    > 周五 18-22 时系统升级，新增 Excel/PDF 导出，修复筛选异常。
  </Tab>
</Tabs>

<Tip>
  如果某个版本还不够满意，可以继续指导 Claude 微调："正式版的第一句太生硬了，改得亲切一点"。
</Tip>
