## 背景

老板让你出一份数据分析报告，要有图表、有结论、有建议。以前你需要：在 Excel 里做数据透视表 → 做图表 → 截图 → 粘到 Word/PPT 里 → 写分析文字。整个流程繁琐且耗时。

<Info>
  **Claude Code 的独特之处**：网页版 AI 可以分析数据、给出文字结论，但无法生成真实的图表文件。Claude Code 会在你的电脑上运行 Python 脚本，生成实际的图表图片和完整的报告文件。
</Info>

## 推荐话术

### 完整分析报告

```
请读取 sales_2024.csv，完成以下任务：

1. 数据概览：总记录数、时间范围、关键指标汇总
2. 趋势分析：按月统计销售额，生成折线图（保存为 charts/月度趋势.png）
3. 结构分析：按产品类别统计占比，生成饼图（保存为 charts/品类占比.png）
4. 排名分析：Top 10 客户销售额，生成柱状图（保存为 charts/Top客户.png）
5. 将以上图表和分析结论整合成一份完整报告，保存为 销售分析报告.md

报告要求：
- 每个分析配一段 3-5 句的文字解读
- 最后附上 3 条可操作的业务建议
- 图表使用中文标签
```

### 同比/环比分析

```
请读取 本月业绩.xlsx 和 上月业绩.xlsx，完成：
1. 计算每个指标的环比变化率
2. 找出变化超过 10% 的指标，标记为"显著变化"
3. 生成一张对比柱状图（保存为 环比对比.png）
4. 写一段 200 字以内的分析摘要，保存为 环比分析.md
```

### 多维度交叉分析

```
请读取 订单数据.csv，按以下维度分析：
1. 按地区 × 产品类别 生成交叉表和热力图（保存为 charts/地区产品热力图.png）
2. 按月份 × 渠道 统计订单量趋势（保存为 charts/渠道月度趋势.png）
3. 找出表现最好和最差的 地区+产品 组合
4. 整理成报告保存为 交叉分析报告.md
```
