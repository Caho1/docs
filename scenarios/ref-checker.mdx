## 背景

很多期刊和基金申请要求论文的参考文献"至少 5 篇、来自 3 个及以上国家"。手动核查的流程极为繁琐：

1. 从 PDF 里把参考文献一条条复制出来
2. 逐篇去 Google Scholar 或数据库查找作者单位
3. 手动记录每位作者的国籍，汇总统计

一篇论文几十条参考文献，光查询就要花半小时以上，还容易漏查。

</Info>

## 工具功能

最终搭出来的工具只需三步：

```
上传论文 PDF  →  自动识别全部参考文献  →  查询每篇文献的作者来源国并汇总
```

结果页面会显示：
- 是否满足"≥5 篇、来自 ≥3 个国家"的要求
- 每条参考文献对应哪些国家
- 所有涉及国家的完整列表

## 实现原理

Claude Code 搭建的工具由三层组成：

```
前端（HTML/Tailwind）
    ↓ 上传 PDF
后端（Python / FastAPI）
    ↓ ① OCR：调用 PaddleOCR API，将 PDF 转为 Markdown
      ② 解析：从 Markdown 中识别参考文献区域，逐条提取
      ③ 查询：调用 OpenAlex API，查询每篇文献的作者所属国家
    ↓ 返回 JSON 结果
前端渲染结果卡片
```

整个过程完全自动，用户只需上传 PDF，等待 30 秒到 2 分钟即可看到结果。

    ```python
    @app.post("/api/analyze")
    async def analyze_pdf(file: UploadFile = File(...)):
        pdf_bytes = await file.read()

        # Step 1：OCR 识别 PDF
        try:
            markdown = await pdf_to_markdown(pdf_bytes)
        except Exception as e:
            return {"success": False, "error": f"OCR 识别失败：{e}"}

        # Step 2：提取参考文献条目
        refs = extract_references(markdown)
        if not refs:
            return {"success": False, "error": "未识别到参考文献"}

        # Step 3：并发查询 OpenAlex
        sem = asyncio.Semaphore(5)
        async with httpx.AsyncClient(timeout=30) as client:
            tasks = [analyze_one_reference(client, ref, i+1, sem)
                     for i, ref in enumerate(refs)]
            results = await asyncio.gather(*tasks)

        return {
            "success": True,
            "total_refs": len(refs),
            "all_countries": sorted(all_countries),
            "meets_requirement": len(all_countries) >= 3 and len(refs) >= 5,
            "references": results,
        }
    ```

    ```python
    def extract_references(markdown: str) -> list[str]:
        # 定位参考文献区域（支持中英文标题）
        HEADERS = ["references", "bibliography", "参考文献"]
        best_pos = max(markdown.lower().rfind(h) for h in HEADERS)
        ref_section = markdown[best_pos:] if best_pos != -1 else markdown[-30%:]

        # 按编号格式逐条分割：[1] 或 1. 或 (1)
        refs = []
        for line in ref_section.split("\n"):
            if re.match(r'^\s*(\[\d+\]|\d+[\.\)])', line):
                refs.append(re.sub(r'^\s*\S+\s*', '', line))
        return refs
    ```
